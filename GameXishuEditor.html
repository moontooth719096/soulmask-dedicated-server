Soulmask GameXishu 編輯器

中文介面、即時搜尋、匯入後編輯，再另存成新的 JSON。適合快速調整大量倍率與開關設定。

匯入 JSON
尚未匯入檔案
你可以直接載入內建預設，再切換 Casual / Easy / Normal / Hard / Master；也可以匯入自己的 JSON 再另存新檔。
難度預設
選擇難度後，編輯器會把對應的預設值套用到目前載入的資料。
搜尋欄位
操作
請先匯入你的 GameXishu.json。
說明
1. 先匯入你自己的 GameXishu.json。
2. 左側切換第 1 / 2 / 3 組設定。
3. 右側先選 休閒 / 簡單 / 標準 / 困難 / 大師，再套用難度預設。
4. 編輯完成後用「另存新檔」下載新 JSON。
5. 0 / 1 類開關會盡量顯示成切換按鈕。
6. 搜尋會同時比對原始欄位名與中文描述。
